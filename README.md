# Sistema di Gestione per Ristorante

Progetto realizzato per il corso di **Ingegneria del Software**.

## Descrizione

Il progetto consiste nello sviluppo di un sistema software per la gestione delle principali attività operative di un ristorante.

L'applicazione consente di gestire:

* ordinazioni dei clienti;
* comunicazione tra sala e cucina;
* gestione del menù;
* gestione del magazzino e dei fornitori;
* prenotazione e gestione della sala eventi.

## Processo software

Il progetto è sviluppato seguendo il **Unified Process (UP)**, adottando un approccio iterativo e incrementale.

##   Casi d'uso principali

* Gestione ordinazioni
* Gestione magazzino e fornitori
* Gestione sala eventi
